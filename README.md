# 🥗 Leftover Rescue - Meal Planner App

A smart, web-based recipe finder that helps users reduce food waste by finding delicious meals they can cook *right now* using only the random ingredients left in their fridge.

## ✨ Features

- **🧊 Fridge Inventory:** Add ingredients as clean, dismissible tags with a sleek interface
- **🔍 Smart Search:** Queries the Spoonacular API to find recipes based on your specific leftovers
- **🛒 Shopping Saver:** Highlights "missing ingredients" in red so you know exactly what you need to buy
- **📱 Responsive Design:** Beautiful on desktop, tablet, and mobile devices
- **⚡ Real-time Updates:** Instant recipe filtering as you add/remove ingredients

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6 Fetch API)
- **Styling:** Modern CSS with animations and gradients
- **API:** Spoonacular API

## 🚀 Getting Started

1. Clone the repository
2. Get a free API key from [Spoonacular](https://spoonacular.com/food-api)
3. Update the API key in `main.js`
4. Open `index.html` in your browser

## 💡 How to Use

1. Type an ingredient name in the input field
2. Press Enter or click "Add Ingredient"
3. Click "Find Recipes" to discover what you can cook
4. Browse recipes with missing ingredients highlighted in red
5. Remove ingredients by clicking the ✕ on their tags
